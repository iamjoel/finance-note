# 控制消费支出小技巧
1. 记录支出。 分门别类记录每笔支出，并且在每比支出后面加上消费类别：固定支出，可降低支出（指可以用更低的价格的），冲动支出（可以不买的）。
1. 分析记录。记录了1个月后，分析支出的大头是什么，那些支出有多大的压缩空间。
1. 制定合理的每月支出计划。 每次要产出冲动消费时，看看是否会超支~
